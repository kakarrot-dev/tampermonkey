# 功能规格：X 阅读增强（CSS 第一阶段）

## 背景

X 默认三栏布局中，右侧推荐栏占用横向空间，中央信息流固定约 600px，阅读密度一般。需要个人油猴脚本做纯视觉增强。

## 目标

1. 隐藏右侧推荐栏（`aside`）及旧版 `sidebarColumn`。
2. 加宽中央信息流（`main` / 旧版 `primaryColumn`），保留左侧导航。
3. 优化帖子间距、正文行高、媒体圆角。
4. 覆盖首页、个人主页、帖子详情、搜索、通知（同一套布局壳）。

## 非目标

- 不修改业务数据、不调用私有接口
- 不自动关注 / 点赞 / 转发
- 不做广告识别增强、主题换色、隐藏左导航
- 第一阶段不做 JS DOM 重排（仅 `GM_addStyle`）

## 用户场景

我在桌面浏览器用 Tampermonkey 打开 x.com，希望信息流更宽、更少干扰，左右导航仍可用。

## 需求边界

包含：CSS 布局与阅读样式；新旧 UI 选择器兼容。  
不包含：登录态自动化、网络拦截、配置面板。

## 匹配范围

```text
@match https://x.com/*
@match https://twitter.com/*
```

## 交互流程

安装脚本后刷新页面即生效；站内 SPA 跳转后样式保持。无额外 UI 控件。

## 数据与状态

无存储。

## 权限与安全

```text
@grant GM_addStyle
```

不碰登录态与隐私数据。

## 异常情况

- 未登录时搜索/通知/首页可能只有登录墙：样式无三栏可作用，属预期。
- X 改版导致 class token / `main`/`aside` 结构变化：按 `docs/selectors.md` 重测更新。

## 验收标准

- 1280 / 1440 / 1920 下右栏隐藏、左导航仍在、主栏约 920px 量级
- 帖子间距 / 行高 / 图片圆角可见改善
- 个人主页 → 详情跳转后样式仍在
- 控制台无脚本报错；无自动互动逻辑
